# Kalaba UI v5 RU

Полностью русская демонстрационная версия MVP Kalaba.

## Основной сценарий

Главная → Поиск → Карточка компании → Заявка → Предложения → Чат → Сделка → Оплата → Отзыв → Личный кабинет.

## Разделы

- Заказчик
- Компания
- CRM
- Финансы
- Аналитика
- Админ-панель
- Модерация
- Поддержка

## Публикация

Замените в GitHub файлы:

- index.html
- styles.css
- README.md

Затем нажмите Commit changes.
