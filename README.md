# Kalaba UI Prototype

Это визуализация приложения Kalaba глазами пользователя.

## Как открыть

1. Распакуйте архив.
2. Откройте `index.html` в браузере.

## Экраны

- Главная
- Поиск услуг
- Карточка компании
- Создание запроса
- Предложения компаний
- Сделка
- Кабинет компании / CRM
